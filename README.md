# 📊 Sales Performance Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Sales Performance Dashboard** built in **Microsoft Power BI** to analyze sales trends, customer performance, product performance, and regional sales insights.

The dashboard enables users to monitor key business metrics and make data-driven decisions through interactive visualizations and filters.

---

## 🎯 Objectives

* Analyze overall sales performance.
* Identify top-performing products and customers.
* Monitor sales trends over time.
* Compare sales across countries and territories.
* Analyze order status and deal size distribution.
* Provide an interactive dashboard for business decision-making.

---

## 📂 Dataset

The dataset contains historical sales transactions with the following fields:

* Order Number
* Quantity Ordered
* Price Each
* Order Line Number
* Sales
* Order Date
* Status
* Quarter
* Month
* Year
* Product Line
* MSRP
* Product Code
* Customer Name
* Phone
* City
* State
* Postal Code
* Country
* Territory
* Deal Size

## 📈 Key Business Insights

The dashboard helps answer questions such as:

* What is the total sales revenue?
* Which product line generates the highest sales?
* Who are the top customers?
* Which countries contribute the most revenue?
* How do sales change over time?
* What is the distribution of deal sizes?
* What percentage of orders are shipped, cancelled, or on hold?
